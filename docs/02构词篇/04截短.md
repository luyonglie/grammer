# 截短  Clipping  

  

>   将单词缩写，词义和词性保持不变的英语构词法称为截短法，主要有截头、去尾、截头去尾 首字母缩略法等形式。  

  

[TOC]



##   1.截头  

| airplane  | plane飞机 |
| --------- | --------- |
| telephone | phone电话 |

  

##   2.去尾  

| kilogram    | kilo千     |
| ----------- | ---------- |
| mathematics | math数学   |
| examination | exam考试   |
| laboratory  | lab实验室  |
| taxicab     | taxi出租车 |

 

##   3.截头去尾  

| prescription | script脚本    |
| ------------ | ------------- |
| influenza    | flu流行性感冒 |
| influenza    | fridge冰箱    |

 

##   4.首字母缩略法  

>   用单词首字母组成一个新词的英语构词法叫做首字母缩略法。这种形式的英语构词生成的新词，读音主要有两种形式，即各字母分别读或作为一个单词来读。
